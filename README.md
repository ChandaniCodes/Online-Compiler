## 🖥️ Online Compiler (Live Code Editor)

This project is a simple **online code editor** that allows users to write and instantly preview code in real time. It provides a split-screen interface where users can type code on one side and see the output on the other.

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* jQuery
* Bootstrap

## ⚙️ How It Works

* Users write code in the **input editor section**
* On every key press, the script captures the content
* The code is dynamically injected into the output section
* The result is displayed instantly on the right side
